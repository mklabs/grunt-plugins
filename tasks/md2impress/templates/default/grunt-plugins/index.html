  • Grunt currently has preliminary plugin support

  • But is already great at packaging a set of tasks / wrapping a "custom" grunt binary

Grunt Plugins

A way to package a set of tasks that get referenced in grunt.js when run via grunt.

A custom global binary wrapping a specific version of grunt, plus some specific extra stuff.

Grunt Plugins

  • grunt-css: Grunt plugin for linting and minifying CSS
  • grunt-less: Grunt task for LESS
  • grunt-bbb: Backbone Boilerplate framework tool.
  • gruntExamples: grunt.js examples for me or someone else
  • build-taks: A collection of useful grunt build tasks

Tasks

A plugin will most likely need to do something useful

This is usually done by defining a new task. Plugins are a great way to package tasks and helpers that can be installed and managed through npm. 

// Load tasks from "grunt-sample" grunt plugin installed via Npm.
grunt.loadNpmTasks("grunt-sample");

Tasks

The tasks/ directory must be used, and mimics grunt's internal directory structure.

When loadNpmTasks is used (typically from within a gruntfile), Grunt will load the tasks in node_modules/grunt-plugin/tasks and make them availale in your grunt setup.

  • Some fancy ASCII Art

Configuration

grunt.js 

module.exports = function(grunt) {
  grunt.initConfig("ascii", {
    color: "bold"
  });

  // Load local tasks.
  grunt.loadTasks("tasks");

  // Default task.
  grunt.registerTask("default", "sourcemap");
};

Tasks

var fs = require('fs'),
  path = require('path');

module.exports = function(grunt) {
  grunt.registerTask('ascii', 'Fancy ascii art', function() {
    var cb = this.async();
    var rs = fs.createReadStream(path.join(__dirname, '../lyonjs.txt')).on('end', cb);
    var colors = Array.prototype.slice.call(arguments);
    // no color! raw stdout
    if(!colors.length) return rs.pipe(process.stdout);
    // color was set! listen for incoming data
    rs.setEncoding('utf8');
    rs.on('data', function(c) {
      // apply each color, rainbow lyonjs is rainbow!
      colors.forEach(function(color) {
        console.log(c[color]);
      });
    });
  });
};
